از آنجایی که شگاه نوکیا به روی ایرانیان بسته است گاهی اوقات لازم می شود که برای دریافت برنامه ای بوسیلهی کامپیوتر برنامه را دریافت کنیم لذا برای این منظور ابتدا باید افزونه ی user agent را برای فایرفاکس خود نصب نمایید سپس به سایت زیر رفته
http://www.developer.nokia.com/
و در قسمت بالا  سمت راست آن مدل گوشی خود را جستجو نمایید.حال صفحه مربوط به گوشی خود را باز نمایید در قسمت Browser, Flash and Web Runtime عبارت روبروی   Example User Agent Headerرا کپی نمایید.سپس از منوی فایرفاکس مسیر Tools>>Default User Agent>>Edit User Agentرا بیاورید و new>>new user agent را کلیک کنید در رویروی  user agent  متن کپی شده را قرار دهید. حال روبروی discription نام مدل گوشی خود را تایپ نمایید.حال ok را کلیک کنید. 
حال به عنوان مثال مسیرTools>>Default User Agent>>N82درمنو انتخاب نمایید که به جای N82 مدل گوشی شما قرار می گیرد.
حال به سایت http://store.ovi.mobi/ مراجعه کرده و نرم افزار دلخواه خود را دریافت نمایید.
